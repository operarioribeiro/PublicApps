# [Public Apps](https://github.com/operarioribeiro/PublicApps)

> _"There is no internet without a true digital public infrastructure (DPI)."_

## Open Project:
**Public Apps** is a [rule of law(RoL)](https://en.wikipedia.org/wiki/Rule_of_law) [open source semantic web ontology](https://www.academia.edu/145734647/Open_Source_Semantic_Web_Ontology_OSSWO_) [Digital Public Infrastructure (DPI)](https://en.wikipedia.org/wiki/Digital_public_infrastructure) [policymaking](https://en.wikipedia.org/wiki/Policymaking) [open standard](https://en.wikipedia.org/wiki/Open_standard) [secure operational development (DevSecOps)](https://en.wikipedia.org/wiki/DevOps#DevSecOps,_shifting_security_left) for [free libre open source software (FLOSS)](https://www.gnu.org/philosophy/floss-and-foss.en.html) [public](https://en.wikipedia.org/wiki/Public_good) [applications](https://en.wikipedia.org/wiki/Application_software).

## Open Methodology:

1. Understand the [open source culture](https://academia.edu/130285565/Open_Source_Culture) history.
2. Study the ['Public Apps: Digital Public Infrastructure (DPI) Free Libre Open Source Software (FLOSS) Applications Applications'](https://www.academia.edu/158815102/Public_Apps) whitepaper.
- Comply with the [public apps](https://dx.doi.org/10.17504/protocols.io.3byl48xo8vo5/v1) protocol. 
- Create your local [open internet](https://www.academia.edu/143169562/Open_Internet) framework.
